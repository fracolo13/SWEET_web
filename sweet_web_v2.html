About SWEET

Synthetic Waveform Emulator for Earthquake Testing

SWEET is an interactive web application for finite fault modeling and synthetic seismic waveform generation. The tool allows users to design custom earthquake rupture scenarios through an intuitive multi-step workflow, from fault geometry definition to kinematic parameter specification, subsource grouping, and spatial station setup.

Key Features

  • Interactive fault geometry design with plane or listric modes
  • Customizable kinematic parameters (slip, rake, rise time, rupture velocity)
  • Moment-based subsource grouping for waveform synthesis
  • Real-time visualization of fault model, slip distribution, and source time functions
  • Spatial station configuration for waveform computation

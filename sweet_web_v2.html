About SWEET

Synthetic Waveform Emulator

SWEET is an interactive web app for finite fault modeling and synthetic seismic waveform generation. Design earthquake rupture scenarios through a map-first workflow: set parameters directly on the map, inspect the fault plane kinematics in an immersive face-on view, group subsources, define a station network, and compute broadband synthetic waveforms.

1. Rupture Setup 2. Kinematics Plane 3. Source Grouping 4. Stations 5. Waveforms
